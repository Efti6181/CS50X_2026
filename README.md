## 🎯 Final Project

### 🔹 Deadlock Detection Simulator with Graph Visualization

A web-based application that demonstrates how deadlocks occur in operating systems using a Resource Allocation Graph.

### ✨ Features:
- Detects deadlocks using DFS cycle detection  
- Provides safe execution sequence when no deadlock is present  
- Interactive graph visualization of processes and resources  
- Stores data using SQLite
